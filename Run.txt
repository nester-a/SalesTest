1. Для запуска выбираем проект веб-апи в качестве автозапускаемого.
2. Нажимаем кнопку запуск
3. У контроллера два метода: получить информацию и совершить продажу. 
4. С получением инфы всё понятно.

5. Метод совершения продажи принимает объект SaleModel, необходимые товары хранятся в нём в словаре, где Key Id продукта, а Value - его количество.
Выглядит он так:
{
  "salesPointId": 0,            <---- идентификатор точки продажи
  "buyerId": 0,                 <---- идентификатор покупателя
  "productsToBuy": {            <---- словарь покупаемых товаров
    "additionalProp1": 0,       <---- элементы словаря
    "additionalProp2": 0,
    "additionalProp3": 0
  }
}

6. В свагере в словарь Id передаётся в ковычках (пример "1":1 - это значит что требуется товар с id 1 в количестве 1 штука), в случае если товаров несколько они передаются черед запятую.
7. После последнего элемента словаря запятая НЕ СТАВИТСЯ
8. В случае если покупателя нет в базе данных, он в методы опускается.
Выглядит это так:
{
  "salesPointId": 0,            <---- идентификатор точки продажи
  "productsToBuy": {            <---- словарь покупаемых товаров
    "additionalProp1": 0,       <---- элементы словаря
    "additionalProp2": 0,
    "additionalProp3": 0
  }
}
